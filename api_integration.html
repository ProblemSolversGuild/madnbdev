Role

 Role (value, names=None, module=None, qualname=None, type=None, start=1)

This corresponds to the roles that openai allows for the ChatGPT API

Role.SYSTEM: "system"
Role.USER: "user"
Role.ASSISTANT: "assistant"
Role.FUNCTION: "function"
